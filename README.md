## build and run test code 

    $ gradle
    $ java -cp build/libs/painter.jar org.notehub.ntwo.painter.test.Test

and then , two png files genereated. so check  

- honeycombAsOne.png
- honeycombAsTwo.png


## usage

see javadoc ( build/docs/javadoc/index.html ) .

- Java 6
